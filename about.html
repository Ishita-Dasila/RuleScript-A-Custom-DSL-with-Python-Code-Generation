About RuleScript Compiler

Everything you need to know about the project

What Is RuleScript?

RuleScript is a custom domain-specific language (DSL) compiler built to simplify the creation of business rule logic. It takes human-readable rule definitions written in a simple RULE / IF / THEN / ELSE syntax and compiles them into clean, executable Python functions.

Instead of writing complex Python code manually, you write intuitive rules like RULE check_age IF age >= 18 THEN eligible = True and the compiler handles the rest — tokenizing, parsing, analyzing, and generating production-ready Python code automatically.

How It Works — The 5 Phases

1

Lexical Analysis (Tokenizer)

Reads your source code and breaks it into tokens — keywords, identifiers, numbers, operators, and booleans.

2

Syntax Analysis (Parser)

Checks that tokens follow the correct grammar order. Raises an error if the structure is wrong.

3

AST Construction

Builds an Abstract Syntax Tree representing the logical structure of each rule.

4

Semantic Analysis

Checks the meaning — detects duplicate rule names and undefined variables. Issues warnings only.

5

Code Generation

Converts the AST into clean Python functions, ready to run or integrate into any project.
