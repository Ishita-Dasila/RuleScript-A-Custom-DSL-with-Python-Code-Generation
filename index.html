What Is RuleScript?

RuleScript is a custom domain-specific language (DSL) compiler built to simplify business rule logic. It allows users to write rules in a human-readable format using simple RULE / IF / THEN / ELSE syntax.

Instead of writing complex Python code manually, users can define logic like:

RULE check_age IF age >= 18 THEN eligible = True

The compiler automatically processes these rules through multiple phases and generates clean, executable Python functions.

It is designed for both developers and beginners, making rule creation faster, easier, and more intuitive.

How It Works — The 5 Phases

1. Lexical Analysis

Breaks input into tokens such as keywords, variables, and operators.

2. Syntax Analysis

Validates the structure of rules according to grammar.

3. AST Construction

Creates a tree structure representing rule logic.

4. Semantic Analysis

Checks meaning, variable usage, and logical correctness.

5. Code Generation

Generates clean Python functions from the AST.

This pipeline ensures accuracy, efficiency, and clarity in rule execution.
